# NGS Variant Calling Pipeline

This repository contains a practical NGS analysis pipeline developed for training and academic purposes.

## Workflow
1. Quality control (FastQC)
2. Read trimming (Trimmomatic)
3. Alignment to reference genome (BWA)
4. BAM processing (SAMtools)
5. Variant calling (bcftools)

## Tools Used
- FastQC
- Trimmomatic
- BWA
- SAMtools
- bcftools

## Input
- Paired-end FASTQ files (not included due to large size)

## Output
- Filtered VCF files
- QC reports
